# resource-management-system
This project is a simple **Resource Management System** (RMS) built in PHP. It allows tracking, organizing, and managing resources efficiently.

## Project Structure

resource-management-system/
│

├── public/ # Publicly accessible files (index.php, assets)

├── src/ # PHP source code

├── vendor/ # Composer dependencies (auto-generated)

├── composer.json # Composer configuration

├── composer.lock # Composer lock file

└── README.md # Project documentation



## Installation

1. Clone the repository:

```bash
git clone https://github.com/misiakfilip/resource-management-system.git
cd resource-management-system
```
2. Install dependencies via Composer:

```bash
composer install
```

3. Make sure your web server points to the public/ directory. For example, in XAMPP, place the project in htdocs and access it via http://localhost/resource-management-system/public.

Usage
All PHP classes in src/ are autoloaded via Composer under the namespace ResourceManagementSystem.

Requirements
- PHP 7.4+ (or 8.x)
- Composer
- PHP extensions required by dompdf: mbstring, dom, gd
- Web server (XAMPP, Apache, Nginx, etc.)

Notes
- The vendor/ directory is generated by Composer and should not be included in Git.
- Use composer install after cloning the repository to generate the vendor/ folder.
- Namespace of classes: ResourceManagementSystem
